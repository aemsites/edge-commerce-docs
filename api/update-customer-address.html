Customers

Update an address

PUT /{org}/sites/{site}/customers/{email}/{subroute}/{resourceId}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Replaces a single address (`subroute=addresses`). Requires ownership of the email.

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
emailstringyesCustomer email address.
subroute"addresses" | "orders"yesEither `addresses` or `orders`.
resourceIdstringyesAddress id or order id.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.
Content-TypestringyesMust be `application/json`.

Request body

FieldTypeRequiredDescription
namestringyesFull name of the recipient.
companystringnoCompany name.
address1stringyesPrimary street address line.
address2stringnoSecondary address line (apartment, suite, etc.).
citystringyesCity name.
statestringyesState or province code.
zipstringyesPostal or ZIP code.
countrystringyesISO 3166-1 alpha-2 country code.
phonestringnoPhone number.
emailstringyes
isDefaultbooleannoWhether this is the default address for the customer.
isValidatedbooleannoWhether this address has been validated by an address verification service.

200 The updated address.

400 The address failed schema validation.

FieldTypeRequiredDescription

errorsobject[]noPer-field validation failures.

FieldTypeRequiredDescription
