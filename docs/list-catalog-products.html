Catalog

List catalog products

GET /{org}/sites/{site}/catalog

Returns a page of product summaries (sku, name, path). Pass a `cursor` (from a prior truncated response) to page through results.

Required permissions: catalog:read

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.

Response

200 A page of product summaries.

truncatedbooleanyesWhether more results are available beyond this page.
cursorstringnoOpaque cursor for the next page; present only when truncated.
