Auth

Log out and clear the session cookie

POST /{org}/sites/{site}/auth/logout

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
AuthorizationstringnoOptional bearer token to revoke. `Bearer <jwt>`.
CookiestringnoOptional `auth_token` cookie to revoke.

Response

204 Logged out (cookie cleared). No content.

HeaderDescription
Set-Cookieauth_token=; Max-Age=0; Path=/; HttpOnly; Secure; SameSite=Strict
