Estimate

Compute an estimate

POST /{org}/sites/{site}/estimate/{estimateType}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Request shape depends on `estimateType`. All variants accept a partial shipping address and (for price/shipping/order) a cart `items` array and optional `couponCode`. An optional BCP-47 `locale` (e.g. `fr-CA`) selects the localized shipping-method label column; the base label is used when it is absent or has no matching column.
