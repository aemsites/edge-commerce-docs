Indices

Delete a query index

DELETE /{org}/sites/{site}/index/{path}

Deletes the query index for the given path and removes it from the registry.

Required permissions: index:write

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
pathstringyesIndex path, must end with `/index.json` (e.g. `/us/index.json`).

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.

Response

204 Index deleted (no content).

400 Path is missing or does not end with `/index.json`.

FieldTypeRequiredDescription
codestringyesMachine-readable error code.
messagestringyesHuman-readable error message.
errorsobject[]noPer-field validation failures.

404 No index exists at the given path.
