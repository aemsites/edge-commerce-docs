Catalog

Bulk write products

POST /{org}/sites/{site}/catalog

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Unified bulk catalog endpoint. Accepts an `{ items }` envelope and, by default, creates or updates (upserts) each product (max 50 per request). Every item is validated up front, so a single invalid product rejects the whole request and saves nothing. Always responds with a `results` array, one entry per item, including for single-item requests. A `delete: true` flag is reserved for bulk deletion in a future release.

Required permissions: catalog:write
