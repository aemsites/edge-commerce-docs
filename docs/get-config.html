Config

Retrieve site configuration

GET /{org}/sites/{site}/config

Returns the stored configuration document for the org/site.

Required permissions: config:read

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.
