Catalog

Delete a product

DELETE /{org}/sites/{site}/catalog/{path}

Deletes the product at the given path. The literal `/*` path is rejected.

Required permissions: catalog:write

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
pathstringyesProduct path (e.g. `/us/en/products/blender-pro-500.json`), or the literal `*` for bulk writes.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.

Response

200 Deletion results.

HeaderDescription
Content-Typeapplication/json

400 Wildcard delete attempted, or malformed path.

detailsobjectnoOptional structured detail.
