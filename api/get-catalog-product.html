Catalog

Retrieve a product

GET /{org}/sites/{site}/catalog/{path}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Returns the stored ProductBus entry for the given path.

Required permissions: catalog:read

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
pathstringyesProduct path (e.g. `/us/en/products/blender-pro-500.json`), or the literal `*` for bulk writes.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.
