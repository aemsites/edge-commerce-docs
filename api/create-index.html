Indices

Create a query index

POST /{org}/sites/{site}/index/{path}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Creates a query index for the given path. The index path is the directory portion of the URL (the trailing `/index.json` is stripped). No request body.

Required permissions: index:write

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
pathstringyesIndex path, must end with `/index.json` (e.g. `/us/index.json`).

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.

Response

201 Index created (no content).
