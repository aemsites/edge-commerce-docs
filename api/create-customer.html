Customers

Create a customer

POST /{org}/sites/{site}/customers

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Validates and stores a new customer. Often called unauthenticated during checkout.

Required permissions: customers:write

Path parameters

sitestringyesSite identifier.

400 The customer failed schema validation.
