Auth

Request an OTP login code

POST /{org}/sites/{site}/auth/login

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
Content-TypestringyesMust be `application/json`.
X-Recaptcha-TokenstringnoreCAPTCHA Enterprise score-based token (OTP/email-abuse defense).

Request body

FieldTypeRequiredDescription
emailstringyesEmail address to send the OTP code to.
countrystringnoOptional ISO country code (default `us`).
localestringnoOptional locale (default `en-US`).

Response

200 OTP challenge; pass these values to /auth/callback.

HeaderDescription
Content-Typeapplication/json
FieldTypeRequiredDescription
emailstringyes
hashstringyesHMAC of the issued code; must be returned to /callback.
expnumberyesExpiry timestamp (ms epoch) of the OTP code.

400 Missing/invalid email, country, or locale.

FieldTypeRequiredDescription
codestringyesMachine-readable error code.
messagestringyesHuman-readable error message.
errorsobject[]noPer-field validation failures.

401 Auth is not enabled for this org/site and the caller is not an existing admin.
