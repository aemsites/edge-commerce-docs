Orders

Patch order custom data

PATCH /{org}/sites/{site}/orders/{orderId}/custom

Merges the request body into the order custom data. Keys with a null value are removed. Each key and value is limited to 2048 characters.

Required permissions: orders:write, orders:custom:write

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
orderIdstringyesOrder identifier (full or friendly id).

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.
Content-TypestringyesMust be `application/json`.
