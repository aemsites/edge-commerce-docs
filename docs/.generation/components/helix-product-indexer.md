# helix-product-indexer documentation impact rules

Update public docs when changes affect:

- Index configuration, generated index fields, sitemap generation, merchant feed generation, or output URLs.
- Reindexing behavior, trigger behavior, freshness expectations, limits, or troubleshooting guidance.
- Public examples for search, filtering, feeds, or sitemap consumption.

Do not update public docs for internal scheduling, worker, storage, logging, or code organization changes with no visible behavior change.
