Places

Autocomplete, details, or address validation (POST)

POST /{org}/sites/{site}/places/{action}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Same actions as GET, plus `validate`: provide `address.addressLines` (non-empty array) and optional `address.regionCode`. Returns a normalized verdict envelope.

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
action"autocomplete" | "details" | "validate"yesWhich Places operation to invoke.

Request headers

NameTypeRequiredDescription
Content-TypestringyesMust be `application/json`.

Response

200 For `validate`, a normalized envelope (`action`, `formattedAddress`, `addressComponents`, `uspsDeliverable`). For `autocomplete`/`details`, the proxied Google response.
