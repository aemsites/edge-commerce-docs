Catalog

Delete a product

DELETE /{org}/sites/{site}/catalog/{path}

Base URL https://api.adobecommerce.live · Staging https://api-stage.adobecommerce.live

Deletes the product at the given path. Supports `If-Match` / `If-None-Match` for conditional deletes; a precondition miss returns `412` and the product is left in place.

Required permissions: catalog:write

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
pathstringyesProduct path (e.g. `/us/en/products/blender-pro-500.json`), or the literal `*` for bulk writes.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.
If-MatchstringnoConditional request. `"<etag>"` proceeds only if the current eTag matches; `*` proceeds only if the product exists. Returns `412` when unmet.
If-None-MatchstringnoConditional request. `*` proceeds only if the product does not exist (create-only). Returns `412` when the product already exists.
