Auth

Verify an OTP code and obtain a session token

POST /{org}/sites/{site}/auth/callback

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
Content-TypestringyesMust be `application/json`.

Request body

FieldTypeRequiredDescription
emailstringyes
codestringyesThe OTP code the user received by email.
hashstringyesThe `hash` returned by /login.
expnumberyesThe `exp` returned by /login.
