Customers

List customers

GET /{org}/sites/{site}/customers

Returns customers for the org/site. Paginated via an opaque `cursor` when the R2 page is truncated.

Required permissions: customers:read

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.

Request headers

NameTypeRequiredDescription
AuthorizationstringyesBearer token. `Bearer <jwt>`.

Response

200 The list of customers (and an optional pagination cursor).
