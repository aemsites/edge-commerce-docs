Places

Autocomplete or place details (GET)

GET /{org}/sites/{site}/places/{action}

For `autocomplete`, provide `input` (and optional `sessiontoken`, `regioncode`). For `details`, provide `place_id` (and optional `sessiontoken`). The proxied Google response is returned verbatim.

Path parameters

NameTypeRequiredDescription
orgstringyesOrganization identifier.
sitestringyesSite identifier.
action"autocomplete" | "details" | "validate"yesWhich Places operation to invoke.
